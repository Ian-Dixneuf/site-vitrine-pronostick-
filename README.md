#  Pronostick – Site Vitrine

##  Présentation

**Pronostick** est un site vitrine développé dans le cadre de la formation **BTS SIO (Services Informatiques aux Organisations)**.

Le projet présente une plateforme fictive de **pronostics sportifs à caractère parodique**.
L’objectif est de concevoir un site vitrine permettant de présenter le concept, les fonctionnalités et l’univers du projet Pronostick.

Ce projet a été réalisé dans un **objectif pédagogique**, afin de mettre en pratique les bases du développement web ainsi que l'utilisation d'outils de gestion de version.

---

##  Objectifs pédagogiques

Ce projet vise à :

* Mettre en pratique les fondamentaux du **développement web**
* Concevoir et structurer un **site vitrine**
* Apprendre à organiser un **projet web**
* Utiliser **Git et GitHub** pour le versioning
* Développer une interface claire et accessible

##  Installation

1. Cloner le repository :

```
git clone git@github.com:Ian-Dixneuf/site-vitrine-pronostick-.git
```

2. Accéder au dossier du projet :

```
cd site-vitrine-pronostick-
```

3. Ouvrir le fichier `index.html` dans votre navigateur.
